# PLM Part Master Database & Dashboard

## Project Overview

This project simulates a simplified Product Lifecycle Management (PLM) Part Master Database using Microsoft Excel.

The solution tracks engineering parts, revisions, ownership departments, lifecycle statuses, and provides interactive analytics through Pivot Tables, Charts, and Dashboard KPIs.

## Features

- Part Master Repository
- Revision Tracking
- Lifecycle Status Management
- Department Ownership Analysis
- Interactive Dashboard
- Pivot Table Reporting

## Excel Skills Used

- Excel Tables
- Data Validation
- Conditional Formatting
- COUNTIF Functions
- Pivot Tables
- Pivot Charts
- Dashboard Design

## PLM Concepts Applied

- Part Master Management
- Revision Control
- Lifecycle Status Tracking
- Engineering Data Analysis
- Department Ownership
